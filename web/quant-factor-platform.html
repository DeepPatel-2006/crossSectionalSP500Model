Capabilities

Data Pipeline

Point-in-time infrastructure

Survivorship-bias-free coverage of roughly 9,000 US equities with strict as-of-date gating on fundamentals, backed by Parquet columnar storage, so every backtest sees only what was knowable at the time.

Factor Validation

Signal, net of costs

Validated a 12-1 cross-sectional momentum factor with a volatility-managed overlay against Ken French's UMD benchmark; characterized additional factors (short-term reversal, profitability) as non-viable once transaction costs were charged.

Machine Learning

A ranking model, honestly tested

Trained a LightGBM stock-ranking model on engineered momentum features. A locked out-of-sample holdout confirmed genuine predictive signal: rank IC +0.019, long-short Sharpe 0.67.

Statistical Engine

Guarding against data-mining

Built a feature-evaluation engine using Newey-West HAC t-statistics, Benjamini-Hochberg FDR correction, decile-spread analysis, and IC-based redundancy clustering, so a factor has to survive multiple-testing correction before it is trusted.

Factor Model Bench · Live

Test factors and build a model, live. Each factor is scored on a synthetic equity panel with the platform's own engine: cross-sectional rank-IC, Newey-West HAC t-stats, decile long-short Sharpe. Check factors to assemble a model and watch the composite's in-sample and out-of-sample performance, cost drag, and factor redundancy update.
